# 周易筮法原理

## 背景

"筮"是传统的周易起卦方式。古人以 50 根蓍草作为占卜工具，名为策，故此法亦称"筹策"占卜。

《周易系辞上传》辞曰：

> "大衍之数五十，其用四十有九。分而为二以像两，挂一以像三，揲之以四以像四时，归奇于扐以像闰，五岁再闰，故再扐而后挂。天一地二，天三地四，天五地六，天七地八，天九地十。天数五，地数五，五位相得而各有合。天数二十有五，地数三十，凡天地之数五十有五。此所以成变化而行鬼神也。"

---

## 大衍之数

整个起卦过程是要求得十八个随机数目，当中包括"六次"（即六根爻）的"三变"。

每爻数值为以下之一：
- **6（老阴）** - 动爻，阴变阳
- **7（少阳）** - 静爻
- **8（少阴）** - 静爻
- **9（老阳）** - 动爻，阳变阴

六爻由下而上排列成本卦（BenGua 本卦）。老阴（6）与老阳（9）为动爻，动爻变化后另成之卦（ZhiGua 之卦）。

### 一变过程

```python
import random

n = 2
stalks_first = 50 - 1          # 把太一拿走
dividers = sorted(random.sample(range(24, stalks_first), n - 1))  # 分二
division = [a - b for a, b in zip(dividers + [stalks_first + 10], [10] + dividers)]
guayi = 1                       # 挂一
right = division[0] - guayi
left_extract = division[1] % 4  # 揲四
if left_extract == 0:
    left_extract = 4
right_extract = right % 4       # 揲四
if right_extract == 0:
    right_extract = 4           # 归奇
bian = left_extract + right_extract + guayi  # 一变结果
```

三变完成后，以 `(49 - 一变 - 二变 - 三变) ÷ 4` 得出爻值（6、7、8 或 9）。

## 处理变爻

参考自【宋】朱熹、蔡元定《易学启蒙》卷下"考变占"：

| 动爻数 | 断法 |
|--------|------|
| 0 爻不变 | 占本卦彖辞，以内卦为贞，外卦为悔 |
| 1 爻变 | 以本卦变爻辞为主 |
| 2 爻变 | 以本卦二变爻辞占，以上爻为主 |
| 3 爻变 | 占本卦及之卦彖辞；初爻变主贞（本卦），二爻起变主悔（之卦） |
| 4 爻变 | 以之卦二不变爻占，以下爻为主 |
| 5 爻变 | 以之卦唯一不变爻占 |
| 6 爻皆变 | 乾坤占二用爻，余卦占之卦彖辞 |

## 纳甲

汉元帝时期，易学名家**京房**（师从焦延寿）开创京氏易学，将筮法融入干支纳甲体系：

以八宫卦为纲，每爻配以天干地支，再结合：
- **五行**：木、火、土、金、水及其生克关系
- **六亲**：父母、官鬼、妻财、兄弟、子孙
- **六兽**：青龙、朱雀、勾陈、螣蛇、白虎、玄武
- **五星**：太白、太阴、岁星、荧惑、填星
- **月建、日辰、二十八宿**

构成完整的六爻象数推演体系，为后世六爻学奠定基础。
